# GitHub User Finder

**Автор:** Артем Марус

## Описание

Программа для поиска пользователей GitHub через официальное API.
Позволяет искать пользователей, просматривать информацию и сохранять в избранное.

## Как получить API

GitHub API не требует ключа. Ограничение: 60 запросов в час.

## Запуск

```bash
python github_user_finder.py
